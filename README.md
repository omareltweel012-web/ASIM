## 🚀 كيفية الاستخدام

1. **افتح الموقع** في متصفحك
2. **ملأ بيانات التسجيل:**
   - الاسم الكامل
   - الرصيد
   - رقم التحويل
   - الضريبة
   - طريقة الدفع
   - رقم المستخدم
3. **اضغط تسجيل** للانتقال للوحة المحفظة
4. **استعرض المحفظة** والخيارات المتاحة
5. **اضغط سحب الأموال** للبدء في عملية السحب
6. **أكمل خطوات السحب** والتأكيد
